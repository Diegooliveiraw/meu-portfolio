Case Study: Sistema de Controle Financeiro

Uma solução Full-Stack para gestão e análise de gastos pessoais.

📌 O Desafio

Centralizar o controle de despesas e receitas de forma simples, mas robusta. O principal problema era criar um sistema que calculasse o saldo em tempo real de forma confiável e que guardasse esses dados com segurança, evitando falhas ou perdas de registros.

💻 A Solução Técnica

Desenvolvi o núcleo da aplicação em Python, responsável pela lógica matemática dos saldos e rotas de dados. Para persistir as informações, integrei um Banco de Dados relacional, garantindo a consistência das transações. A interface foi construída com HTML5, CSS3 e JavaScript, permitindo que a atualização dos gráficos e tabelas de fluxo de caixa aconteça de forma dinâmica e fluida.

🚀 O que aprendi

Este projeto consolidou meu entendimento sobre arquitetura MVC (Model-View-Controller), manipulação de queries SQL e como gerenciar estados no Front-End para que batam exatamente com o que está salvo no Banco de Dados.

Case Study: Dashboard de Clima

Consumo de APIs REST e manipulação assíncrona de dados.

📌 O Desafio

Criar uma interface que entregasse dados climáticos instantâneos sem que a página precisasse recarregar. O desafio técnico era lidar com requisições de rede de forma assíncrona, tratando possíveis erros (como cidades não encontradas ou falhas de conexão) para que o usuário não ficasse sem resposta.

💻 A Solução Técnico

Utilizei a **Fetch API** do JavaScript com a sintaxe moderna de **async/await** para conectar a aplicação à API da OpenWeatherMap. Os dados retornados em formato JSON são tratados e injetados dinamicamente na tela alterando o DOM. Também implementei blocos de tratamento de erro (`try/catch`) para garantir uma experiência de usuário sem quebras (UX fluida).

🚀 O que aprendi

Aprofundei significativamente meus conceitos sobre concorrência no JavaScript, manipulação do ciclo de vida de requisições HTTP (GET) e como componentizar o código para manter funções de busca separadas da lógica de renderização visual.
